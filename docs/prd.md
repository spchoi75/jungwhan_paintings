# PRD: Jungwhan 미술작품 포트폴리오

> **버전**: 1.0
> **작성일**: 2026-01-24
> **상태**: Draft

---

## 1. 개요

### 1.1 제품 정의
미술작가 Jungwhan의 작품을 전시하는 **개인 포트폴리오 웹사이트**.
세련되고 미니멀한 디자인으로 작품 자체에 집중하며, 관리자가 쉽게 작품을 추가/관리할 수 있는 시스템.

### 1.2 핵심 목표
| 목표 | 설명 |
|------|------|
| 작품 전시 | 고해상도 작품을 세련된 갤러리 형태로 전시 |
| 확대 감상 | 작품 디테일을 확인할 수 있는 줌 기능 제공 |
| 쉬운 관리 | 관리자가 코딩 없이 작품 추가/수정/삭제 가능 |
| 세련된 UX | AI가 만든 티가 나지 않는 고급스러운 인터페이스 |

### 1.3 핵심 원칙
- **작품 중심**: UI는 작품을 방해하지 않음
- **미니멀리즘**: 불필요한 요소 배제
- **반응형**: 모바일/태블릿/데스크톱 대응
- **성능**: 이미지 최적화로 빠른 로딩

## 3. 기능 명세

### 3.1 F1: 랜딩 페이지

**설명**: 사이트 첫 진입 화면. 강렬한 첫인상을 주는 히어로 섹션과 대표작 미리보기.

| 항목 | 내용 |
|------|------|
| Input | 페이지 접속 |
| Process | 히어로 이미지 로드, 대표작 썸네일 로드 |
| Output | 풀스크린 히어로 + 대표작 그리드 렌더링 |

**UI 요소**:
- 히어로 섹션: 대표작 1점 풀스크린 배경
- 작가명/타이틀: 미니멀 타이포그래피
- 스크롤 인디케이터: 아래로 스크롤 유도
- 대표작 미리보기: 3~6점 그리드

---

### 3.2 F2: 갤러리 페이지

**설명**: 전체 작품을 그리드 형태로 전시.

| 항목 | 내용 |
|------|------|
| Input | 갤러리 페이지 접속 |
| Process | 작품 목록 조회, 썸네일 렌더링 |
| Output | 작품 그리드 표시 |

**UI 요소**:
- Masonry 또는 균일 그리드 레이아웃
- 호버 시 작품 제목 표시
- 클릭 시 상세 뷰 진입
- Lazy loading 적용

---

### 3.3 F3: 작품 상세 뷰 (모달)

**설명**: 개별 작품을 크게 보고, 확대 감상 가능.

| 항목 | 내용 |
|------|------|
| Input | 작품 클릭 |
| Process | 고해상도 이미지 로드, 메타데이터 표시 |
| Output | 모달 오버레이에 작품 + 정보 렌더링 |

**UI 요소**:
- 작품 이미지 (고해상도)
- 작품 정보: 제목, 연도, 크기, 재료
- 닫기 버튼
- 좌우 화살표 (이전/다음 작품)
- **확대 기능** (핀치줌 또는 클릭줌)

**확대 기능 상세**:
- 데스크톱: 클릭으로 줌 인/아웃, 마우스 드래그로 패닝
- 모바일: 핀치줌, 터치 드래그로 패닝
- 최대 확대 배율: 원본 해상도까지

---

### 3.4 F4: 관리자 페이지

**설명**: 작품 추가/수정/삭제를 위한 관리 화면.

| 항목 | 내용 |
|------|------|
| Input | 관리자 로그인 후 접속 |
| Process | 작품 CRUD 처리 |
| Output | 작품 목록 테이블 + 폼 |

**UI 요소**:
- 작품 목록 테이블 (썸네일, 제목, 연도, 액션)
- 작품 추가 버튼
- 작품 추가/수정 폼:
  - 이미지 업로드 (드래그 앤 드롭)
  - 제목 입력
  - 연도 입력
  - 크기 입력 (가로 x 세로 cm)
  - 재료/기법 입력
  - 대표작 여부 체크박스
- 삭제 확인 모달

---

### 3.5 F5: 관리자 인증

**설명**: 관리자 페이지 접근을 위한 간단한 인증.

| 항목 | 내용 |
|------|------|
| Input | 비밀번호 입력 |
| Process | 비밀번호 검증 |
| Output | 인증 성공 시 관리자 페이지 접근 허용 |

**상세**:
- 단일 관리자 계정 (ID 불필요, 비밀번호만)
- 세션 기반 인증 (브라우저 종료 시 만료)
- 환경변수로 비밀번호 관리

## 4. 데이터 모델

### 4.1 Artwork (작품)

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| id | string | Y | 고유 식별자 (UUID) |
| title | string | Y | 작품 제목 |
| year | number | Y | 제작 연도 |
| width | number | N | 가로 크기 (cm) |
| height | number | N | 세로 크기 (cm) |
| medium | string | N | 재료/기법 (예: "Oil on canvas") |
| imageUrl | string | Y | 원본 이미지 URL |
| thumbnailUrl | string | Y | 썸네일 이미지 URL |
| isFeatured | boolean | Y | 대표작 여부 (기본값: false) |
| order | number | Y | 정렬 순서 |
| createdAt | datetime | Y | 등록일시 |
| updatedAt | datetime | Y | 수정일시 |

## 5. 비기능 요구사항

### 5.1 성능
| 항목 | 목표 |
|------|------|
| 초기 로딩 | 3초 이내 (LTE 환경) |
| 이미지 최적화 | WebP 포맷, 썸네일 자동 생성 |
| Lazy Loading | 뷰포트 진입 시 이미지 로드 |

### 5.2 반응형 브레이크포인트
| 구분 | 너비 |
|------|------|
| 모바일 | < 768px |
| 태블릿 | 768px ~ 1024px |
| 데스크톱 | > 1024px |

### 5.3 브라우저 지원
- Chrome, Safari, Firefox, Edge (최신 2개 버전)
- iOS Safari, Android Chrome

---

## 6. 제외 범위 (Out of Scope)

다음 기능은 이번 버전에서 **구현하지 않음**:

- 다국어 지원
- 작품 검색/필터
- 방명록/댓글
- 작품 판매/문의 기능
- 작가 소개 페이지
- SEO 최적화 (기본 메타태그 외)
- 애널리틱스 연동
- 소셜 공유 버튼

## 8. 검증 기준

### 8.1 기능 검증
| 기능 | 검증 방법 |
|------|----------|
| 랜딩 페이지 | 히어로 이미지 렌더링, 대표작 표시 확인 |
| 갤러리 | 전체 작품 그리드 표시, 클릭 시 상세 뷰 진입 |
| 작품 상세 | 모달 표시, 확대/축소/패닝 동작 확인 |
| 관리자 인증 | 올바른 비밀번호로만 접근 가능 |
| 작품 추가 | 이미지 업로드 후 갤러리에 표시 |
| 작품 수정 | 정보 수정 후 반영 확인 |
| 작품 삭제 | 삭제 후 갤러리에서 제거 |

### 8.2 비기능 검증
| 항목 | 검증 방법 |
|------|----------|
| 반응형 | Chrome DevTools로 각 브레이크포인트 확인 |
| 이미지 로딩 | Network 탭에서 lazy loading 동작 확인 |
| 확대 기능 | 모바일 실기기에서 핀치줌 테스트 |

## 9. 용어 정의

| 용어 | 정의 |
