## 사이트 네비게이션

### 사이드 패널 사용법
1. **열기**: 화면 좌측 상단 햄버거 메뉴 클릭 또는 좌측 세로 핸들 클릭
2. **메뉴 탐색**: 메뉴 항목을 클릭해도 패널이 열린 상태 유지
3. **닫기**:
   - 어두운 배경(오버레이) 클릭
   - X 버튼 클릭
   - 패널 우측 세로 핸들 클릭

### 메뉴 구조
- **Home**: 메인 페이지 (대표작 슬라이드쇼)
- **Portfolio**: 작품 카테고리별 갤러리
- **About**: 작가 소개
- **Exhibitions**: 전시 이력
- **Contact**: 연락처

## 1. 카테고리 관리

### 카테고리 추가
1. `/admin` 접속 → 비밀번호 입력
2. **"카테고리"** 탭 선택
3. **"카테고리 추가"** 버튼 클릭
4. 정보 입력:
   - **카테고리 이름** (필수): 예) 풍경화, 인물화, 추상화
   - **슬러그 (URL)** (필수): 예) landscape, portrait, abstract
     - 자동 생성되며 수정 가능
     - URL에 사용됨: `/artworks/landscape`
   - **설명** (선택): 카테고리에 대한 간단한 설명
   - **커버 이미지** (선택): 포트폴리오 페이지에서 썸네일로 표시
5. **"저장"** 클릭

### 카테고리 수정/삭제
- **수정**: 카테고리 행의 "수정" 버튼 클릭
- **삭제**: 카테고리 행의 "삭제" 버튼 클릭
  - 주의: 해당 카테고리의 작품들은 "미분류"로 변경됨

---

## 2. 작품 관리

### 작품 추가
1. `/admin` 접속 → 비밀번호 입력
2. **"작품"** 탭 선택
3. **"작품 추가"** 버튼 클릭
4. 정보 입력:
   | 필드 | 필수 | 설명 |
   |------|------|------|
   | 이미지 | O | 드래그앤드롭 또는 클릭해서 업로드 |
   | 제목 | O | 작품 제목 |
   | 카테고리 | - | 드롭다운에서 선택 |
   | 연도 | O | 제작 연도 |
   | 가로/세로 | - | 크기 (cm) |
   | 재료/기법 | - | 예) Oil on canvas |
   | 작품 설명 | - | 상세 설명 |
   | 대표작 설정 | - | 체크 시 메인 슬라이드쇼에 표시 |
5. **"저장"** 클릭

### 작품 수정
1. 작품 목록에서 수정할 작품의 **"수정"** 버튼 클릭
2. 내용 수정 후 **"저장"**

### 작품 삭제
1. 삭제할 작품의 **"삭제"** 버튼 클릭
2. 확인 메시지에서 승인

### 대표작 설정
- 대표작으로 설정된 작품은 메인 페이지 슬라이드쇼에 표시됩니다
- 작품 추가/수정 시 "대표작으로 설정" 체크박스 사용

---

## 3. 전시 관리

### 전시 추가
1. `/admin` 접속 → 비밀번호 입력
2. **"전시"** 탭 선택
3. **"전시 추가"** 버튼 클릭
4. 정보 입력:
   | 필드 | 필수 | 설명 |
   |------|------|------|
   | 전시명 | O | 전시 제목 |
   | 장소 | O | 갤러리/기관명 |
   | 지역 | - | 예) 서울, 한국 |
   | 연도 | O | 전시 연도 |
   | 유형 | O | 개인전 / 그룹전 |
   | 외부 링크 | - | 전시 관련 URL |
5. **"추가"** 클릭

### 전시 수정/삭제
- **수정**: 전시 행의 "수정" 버튼 클릭
- **삭제**: 전시 행의 "삭제" 버튼 클릭

---

## 4. 작가 소개 관리

### 작가 정보 수정
1. `/admin` 접속 → 비밀번호 입력
2. **"소개"** 탭 선택
3. 수정 가능한 항목:
   - **프로필 이미지**: 드래그앤드롭 또는 클릭해서 업로드
   - **작가명**: 사이트에 표시되는 이름
   - **소개글**: 여러 문단 추가 가능 (+ 문단 추가 버튼)
   - **Education**: 학력 정보 (연도 + 내용)
   - **Exhibitions**: 전시 이력 (연도 + 내용)
   - **연락처 이메일**: Contact 페이지에 표시
4. **"저장"** 클릭

---

## 5. 비밀번호 변경

### 관리자 페이지에서 변경
1. `/admin` 접속 → 로그인
2. **"설정"** 탭 선택
3. 정보 입력:
   - **현재 비밀번호** (필수)
   - **새 비밀번호**: 변경하지 않으려면 비워둠
   - **새 비밀번호 확인**: 새 비밀번호와 동일하게 입력
   - **힌트 메시지**: 로그인 페이지에서 표시될 힌트
4. **"설정 저장"** 클릭

### 비밀번호 규칙
- 최소 4자 이상
- 숫자, 영문 조합 권장

## 7. Supabase 직접 관리

### 대시보드 접속
https://supabase.com/dashboard → 프로젝트 선택

### 테이블 구조
| 테이블 | 용도 |
|--------|------|
| `artworks` | 작품 정보 |
| `categories` | 카테고리 정보 |
| `exhibitions` | 전시 정보 |
| `about_info` | 작가 소개 정보 |
| `admin_settings` | 관리자 설정 (비밀번호 해시) |

### 데이터 직접 수정
1. **Table Editor** 메뉴
2. 테이블 선택
3. 행 클릭 후 직접 수정

### 이미지 저장소 확인
1. **Storage** 메뉴
2. `artworks` 버킷에서 업로드된 이미지 확인/삭제

## 9. 문제 해결

### 이미지가 안 보여요
1. Supabase Storage 정책 확인
2. 이미지 URL이 올바른지 확인
3. 브라우저 캐시 삭제 후 새로고침

### 작품이 안 나와요
1. Supabase `artworks` 테이블에 데이터 있는지 확인
2. 브라우저 개발자 도구 콘솔에서 에러 확인

### 관리자 로그인이 안 돼요
1. 현재 비밀번호: `123456`
2. 힌트: "숫자 6자리"
3. Supabase `admin_settings` 테이블 확인

### 사이드 패널이 안 열려요
1. 브라우저 새로고침
2. 다른 브라우저에서 테스트
3. JavaScript 활성화 확인
