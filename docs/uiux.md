# UI/UX Design: Jungwhan 미술작품 포트폴리오

> **버전**: 1.0
> **작성일**: 2026-01-24
> **기준 문서**: [PRD](./prd.md), [Architecture](./architecture.md), [WBS](./wbs.md)

---

## 1. 디자인 원칙

| 원칙 | 적용 |
|------|------|
| **작품 중심** | UI는 배경으로 물러나고, 작품이 주인공 |
| **미니멀리즘** | 불필요한 장식 배제, 여백 활용 |
| **고급스러움** | 갤러리/미술관 분위기, AI 티 배제 |
| **직관성** | 설명 없이 사용 가능한 인터페이스 |

---

## 2. 정보 구조 (IA)

```
Home (/)
├── Hero Section (대표작 풀스크린)
└── Featured Works (대표작 3~6점)

Gallery (/gallery)
├── Artwork Grid (전체 작품)
└── Artwork Modal (상세 뷰 + 확대)

Admin (/admin)
├── Login (/admin/login)
└── Dashboard (/admin)
    ├── Artwork List
    └── Artwork Form (추가/수정)
```

## 3. 디자인 시스템

### 3.1 색상 팔레트

```
┌─────────────────────────────────────────────────────┐
│  Primary Colors                                      │
├─────────────────────────────────────────────────────┤
│  ■ Background     #FAFAFA (밝은 회백색)              │
│  ■ Surface        #FFFFFF (순백)                    │
│  ■ Text Primary   #1A1A1A (거의 검정)               │
│  ■ Text Secondary #6B6B6B (중간 회색)               │
│  ■ Border         #E5E5E5 (연한 회색)               │
│  ■ Accent         #2C2C2C (진한 차콜)               │
└─────────────────────────────────────────────────────┘
```

### 3.2 타이포그래피

| 용도 | 폰트 | 크기 | 두께 |
|------|------|------|------|
| 작가명 (Hero) | Cormorant Garamond | 48px / 32px (mobile) | 300 Light |
| 섹션 제목 | Cormorant Garamond | 24px | 400 Regular |
| 작품 제목 | Pretendard | 16px | 500 Medium |
| 본문/정보 | Pretendard | 14px | 400 Regular |
| 버튼/라벨 | Pretendard | 14px | 500 Medium |

### 3.3 간격 시스템

```
4px  (xs)  - 아이콘 내부
8px  (sm)  - 요소 내부 패딩
16px (md)  - 카드 패딩
24px (lg)  - 섹션 간격
48px (xl)  - 페이지 여백
80px (2xl) - 히어로 섹션 여백
```

### 3.4 반응형 브레이크포인트

| 구분 | 너비 | 그리드 컬럼 |
|------|------|------------|
| Mobile | < 768px | 1~2열 |
| Tablet | 768px ~ 1024px | 2~3열 |
| Desktop | > 1024px | 3~4열 |

### 4.3 작품 상세 모달

```
┌─────────────────────────────────────────────────────┐
│                                              [✕]    │
│                                                      │
│   [←]              [작품 이미지]              [→]    │
│                   (확대/축소 가능)                    │
│                                                      │
│                                                      │
├─────────────────────────────────────────────────────┤
│                                                      │
│   작품 제목                                          │
│   2024 · 100 × 80 cm · Oil on canvas                │
│                                                      │
└─────────────────────────────────────────────────────┘
```

| 상태 | 처리 |
|------|------|
| Loading | 블러된 썸네일 → 고해상도 페이드인 |
| Error | "이미지를 불러올 수 없습니다" + 재시도 버튼 |

**인터랙션**:
- 확대: 클릭 또는 핀치
- 패닝: 드래그 (확대 상태에서)
- 닫기: ✕ 버튼, ESC 키, 배경 클릭
- 이동: ← → 버튼, 좌우 화살표 키, 스와이프 (모바일)

**확대 컨트롤**:
```
┌─────────────────┐
│  [−]  [+]  [⟲]  │   ← 줌 아웃, 줌 인, 리셋
└─────────────────┘
```

## 5. 인터랙션 & 애니메이션

### 5.1 트랜지션

| 요소 | 효과 | 지속시간 |
|------|------|----------|
| 페이지 전환 | 페이드 | 200ms |
| 모달 열기 | 페이드 + 스케일 (0.95→1) | 200ms |
| 모달 닫기 | 페이드 + 스케일 (1→0.95) | 150ms |
| 호버 효과 | 오버레이 페이드 | 200ms |
| 버튼 호버 | 배경색 변경 | 150ms |
| 이미지 로드 | 페이드인 | 300ms |

### 5.2 스크롤 인디케이터

```css
@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(8px); }
}
/* 2초 주기, 무한 반복 */
```

### 5.3 확대 제스처

| 플랫폼 | 제스처 |
|--------|--------|
| Desktop | 클릭: 줌 토글, 휠: 줌 인/아웃, 드래그: 패닝 |
| Mobile | 핀치: 줌 인/아웃, 더블탭: 줌 토글, 드래그: 패닝 |

## 6. 접근성

### 6.1 키보드 네비게이션

| 키 | 동작 |
|----|------|
| Tab | 포커스 이동 |
| Enter/Space | 버튼 활성화 |
| ESC | 모달 닫기 |
| ← / → | 작품 이동 (모달) |

### 6.2 ARIA 레이블

```html
<button aria-label="작품 상세 보기: 작품제목">
<button aria-label="이전 작품">
<button aria-label="다음 작품">
<button aria-label="모달 닫기">
<div role="dialog" aria-modal="true" aria-labelledby="modal-title">
```

### 6.3 색상 대비

- 텍스트/배경 대비: 최소 4.5:1 (WCAG AA)
- 포커스 표시: 2px solid #2C2C2C

## 7. 피드백 패턴

### 7.1 로딩 상태

| 요소 | 표시 |
|------|------|
| 페이지 | 스켈레톤 UI |
| 버튼 | 스피너 + 비활성화 |
| 이미지 | 블러 플레이스홀더 → 페이드인 |

### 7.2 성공 피드백

| 액션 | 피드백 |
|------|--------|
| 저장 완료 | 토스트 "저장되었습니다" (2초) |
| 삭제 완료 | 토스트 "삭제되었습니다" (2초) |
| 로그인 성공 | 리다이렉트 |

### 7.3 에러 피드백

| 유형 | 표시 |
|------|------|
| 폼 검증 | 필드 하단 빨간색 텍스트 |
| API 에러 | 토스트 또는 인라인 메시지 |
| 네트워크 에러 | "연결을 확인해주세요" + 재시도 |
